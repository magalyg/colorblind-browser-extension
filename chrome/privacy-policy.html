Privacy Policy

ColorBlind Accessibility Filter — Last updated April 20, 2026

This extension does not collect, store, transmit, or share any personal data.

What this extension does

ColorBlind Accessibility Filter is a Chrome extension that applies color filters to webpages to improve accessibility for colorblind individuals. Users can select from four modes: Deuteranopia, Protanopia, Tritanopia, and Achromatopsia.

Data we collect

We collect no personal data. The only information stored by the extension is your selected color mode, which is saved locally in your browser using Chrome's built-in storage.sync API. This preference:

  • Is stored on your device and optionally synced across your own Chrome browsers via your Google account
  • Contains no personally identifiable information
  • Is never transmitted to any server we operate or any third party

Permissions used

  • storage — saves your selected color mode between sessions
  • activeTab — applies the color filter to the tab you are currently viewing
  • scripting — injects the color filter into tabs that were open before the extension was installed

No permission is used to read, capture, or transmit any content from the pages you visit.

Third parties

This extension does not integrate with any third-party services, analytics platforms, or advertising networks. No data is sold or shared with any outside party.

Remote code

This extension does not load or execute any remote code. All JavaScript is bundled within the extension package.

Children's privacy

This extension does not collect data from anyone, including children under the age of 13.

Changes to this policy
